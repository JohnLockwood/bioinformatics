## Books

Several resources are recommended in this [ChatGPT repsonse](https://chatgpt.com/c/6a37c788-6d84-83ea-9fff-59a9463e51bd) (private link).  Pevzner's [video series](https://www.bioinformaticsalgorithms.org/lecture-videos) is also worth a look.

From the Gemini [deep research link](https://gemini.google.com/app/9d42be88655591f1) two books stand out:

* Vince Buffalo's [Bioinformatics Data Skills](https:/www.amazon.com/Bioinformatics-Data-Skills-Reproducible-Research/dp/1449367372/) has a lot of information that repeats what you know about the Unix command line (and some new stuff), Git, etc., but has a lot of bioinformatics tools and data format info too.  You can get a good feel for the table of content on [OReilly's site](https://www.oreilly.com/library/view/bioinformatics-data-skills/9781449367480/).
* [Orchestrating Single Cell Analysis with Bioconductor](https://bioconductor.org/books/release/OSCA/).  This is a free online book.  Bioconductor is important for the statistical analysis that one needs to do late in the process, where it looks like command line Unix + Python is what are used early in the pipeline.

## Courses and Free Tutorials

The tutorial for the online sandbox [sandbox.bio](https://sandbox.bio) is probably worth going through.  This is a fun tool that starts you off reading FASTA files.  There are also a whole bunch of [great tutorials](https://sandbox.bio/tutorials) on different tools. There of course is the certificate program at UNCC, which would carry an estimated price tag of $5,000 to $7,000. ([Source](https://gemini.google.com/app/abb211788df407b5)).  This is as of 2026.
